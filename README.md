# Inventory Forecasting Analysis
**Demand Forecasting using Time Series (ARIMA)**

This project forecasts future inventory needs using historical sales data and time series techniques.

---

## 🧾 Project Overview
The goal is to optimize stock planning using models that forecast demand for upcoming weeks.

---

## 📊 Dataset
- Features: Date, product ID, past inventory/sales data
- Target: `Next_Inventory_Quantity`

---

## ⚙️ Tools & Libraries
- Python  
- Pandas  
- Scikit-learn  
- Statsmodels (ARIMA)  
- Matplotlib / Seaborn  

---

## 🚀 Models Used
- Prophet  
- ARIMA ✅ *(Final Selected Model)*

---

## ✅ Final Model Performance
- **Selected Model**: ARIMA  
- **RMSE**: ~12.8  
- Best suited for capturing seasonality and trend in time-based inventory data.

---

## 📁 Files
- `Inventory_Forecasting_Final.ipynb`: Full modeling notebook with EDA, ARIMA forecasting, and final output

---

## 📌 Summary
ARIMA performed best for time series prediction and provided interpretable results for demand forecasting.
